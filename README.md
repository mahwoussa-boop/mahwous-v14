# 🌹 مهووس — تطبيق المراجعة البصرية والإضافة

تطبيق متطور مبني باستخدام **Streamlit** و **Python** لإدارة ومراجعة المنتجات، توليد الأوصاف باستخدام الذكاء الاصطناعي، ومطابقة المنتجات بين المتاجر المختلفة.

## 📁 هيكلية المشروع

تم تنظيم المشروع بهيكلية مستقرة تضمن سهولة الإدارة والتوسع:

- `mahwous_app.py`: الملف الرئيسي للتطبيق.
- `data/`: يحتوي على جميع البيانات المدخلة:
  - `our_store/`: ملفات CSV الخاصة بمتجر "مهووس".
  - `competitors/`: ملفات CSV الخاصة بالمنافسين.
  - `helpers/`: ملفات مساعدة (الماركات، التصنيفات).
- `output/`: المجلد المخصص للملفات المصدرة (النتائج).
- `requirements.txt`: ملف المكتبات المطلوبة لتشغيل المشروع.
- `.streamlit/config.toml`: ملف إعدادات Streamlit.

## 🚀 التشغيل المحلي

1. تأكد من تثبيت Python 3.8+.
2. قم بتثبيت المكتبات المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
3. قم بتشغيل التطبيق:
   ```bash
   streamlit run mahwous_app.py
   ```

## ☁️ النشر على Streamlit Cloud

للنشر على Streamlit Cloud، تأكد من أن المستودع عام (Public) وأن ملف `requirements.txt` و `mahwous_app.py` موجودان في الجذر. سيتعرف Streamlit Cloud تلقائياً على التطبيق ويقوم بنشره.

## 🛠️ التقنيات المستخدمة
- **Streamlit**: لواجهة المستخدم التفاعلية.
- **Pandas**: لمعالجة وتحليل البيانات.
- **RapidFuzz**: للمطابقة الذكية بين أسماء المنتجات.
- **Requests**: للتعامل مع واجهات برمجة التطبيقات (APIs).
